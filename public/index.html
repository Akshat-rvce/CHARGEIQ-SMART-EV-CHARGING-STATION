SPECIFICATIONS

Hardware & Software Stack.

Microcontroller

ESP32 DevKit V1

Dual-core 240MHz core, built-in WiFi module, 18-channel 12-bit ADC ports for sensor telemetry mapping.

Local Readout

16x2 LCD with I2C

On-device LCD module updating local battery parameters, current modes, and warning indices.

Thermal probe

NTC Thermistor

Analog thermistor configured via Steinhart-Hart logarithmic calculations for exact cell temperature sensing.

Current Sensor

ACS712 Sensor

Hall-effect current sensing module resolving current flow rates up to 5A with high ADC precision.

Safety Circuit

Relay & Driver

5V active relay driven by a BC547 NPN transistor switch with 1N4007 flyback diode protection.

Mamdani Engine

Fuzzy Logic Controller

Embedded logic using 5 Mamdani fuzzy rules defuzzified by centroid algorithms.
